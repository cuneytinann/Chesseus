# 2K FIDE PromptChess

> **Full FIDE-compliant chess in 2032 bytes.**

All rules. No dependencies. No build step. Both sides played by humans. A small but legible interface: monospaced fullwidth glyphs (`Ｒ Ｎ Ｂ Ｑ Ｋ`) keep the board aligned regardless of the browser's prompt font, the timer displays seconds (`B: 900s        W: 900s`), and any invalid input is silently rejected — the prompt simply reopens.
